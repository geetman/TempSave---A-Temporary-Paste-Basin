# TempSave A Temporary Paste Basin
A temporary paste basin with zero persistence. Open multiple Windows-95-style spaces, paste anything (text, images, links, files), copy it back out when you need it. Nothing gets saved. Pure client-side, pure webcore aesthetic. Close the tab and it all evaporates.

## Features
- Ephemeral Paste Spaces — Open unlimited windows; nothing saves to disk
- Universal Paste — Text, images, links, HTML, tables, files via drag & drop
- One-Click Copy Out — Grab content back to system clipboard instantly
- Multi-Space Workflow — Run parallel clipboards side by side
- Pure Webcore Aesthetic — CRT scanlines, Windows 95/XP chrome, pixelated textures
- Zero Backend — 100% client-side, no accounts, no tracking, no server
- BSOD Easter Egg — Because nostalgia should bite back
- Self-Contained — Single HTML file, runs anywhere offline
